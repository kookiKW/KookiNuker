- You must have Python https://www.python.org/downloads/
- Open the [Releases](https://github.com/kookiKW/KookiNuker.git)Page.
- Click on the latest version.
- Extract the zip.
- Open `Kooki.exe`.
- Enter your user token, bot token, or victims Token.
